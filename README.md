Project predicting Airbnb rental prices from home information.

Contains:
* R code with EDA and model training/performance
* Train (3000+ rows) and test (700 rows) datasets
* Technical and non-technical report summarizing findings
* Testing predictions from best model (random forest)

Tools and Frameworks:
* tidyverse (EDA)
* regclass (checking variance inflation factor)
* leaps (subset selection)
* caret (cross-validation)
* glmnet (ridge/lasso regression)
* tree (decision tree)
* randomForest
